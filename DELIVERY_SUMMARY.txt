USAGE INSTRUCTIONS
================================================================================

Quick Start:
  1. pip install -r requirements.txt
  2. python generate_nfts.py
  3. python verify_collection.py
  4. open gallery.html

Generate 2000 NFTs:
  1. Edit config.json: "collection_size": 2000
  2. python generate_nfts.py

Add Custom Artwork:
  1. Create 1000x1000 PNG files
  2. Name exactly as in config.json
  3. Place in layers/ subdirectories
  4. python generate_nfts.py

View Collection:
  - Open gallery.html in browser
  - Or: python -m http.server 8000
